所谓的存在，being 就像程序中的 = 赋值一样。

不断的 =，直到构建出一个 Object 对象。

究其根本，只有两种，一是理性主义给出的运算功能，比如 int float long double，二是表意符号性的，比如 char boolean String，比如 BigDecimal 就是为了兼备表意性和运算性。

但是代码构建出来的世界是现实世界的全部吗？也许只是维特根斯坦说的可说之物罢了。
