学习本身也是有套路的！

<!-- more -->

刻意练习包含了三个步骤。
第一，找字典：找到你要学习的这个领域体系的范式（pattern）；
第二，找题和答案：针对每个范式刻意的反复学习和练习；
第三，改错：及时反馈。

helloworld 式、[MVP](MVP.md)：先跑起来！比如做博客，先跑起来一个再逐个点去完善，对于逐个点也是，先 run 再精进。这背后其实是一种广度优先的结构化思维。
