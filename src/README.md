在知识管理领域，有一个经典的[DIKW](DIKW.md)模型（数据-信息-知识-智慧金字塔）：　

- [[数据]](Data)：原始的、未经处理的事实和观察结果
- [[信息]](Information)：经过组织和解释的数据，具有上下文和意义
- [[2 第二大脑/1 宇宙概念树/知识]](Knowledge)：通过经验和学习内化的信息，能够指导行动
- [[智慧]](Wisdom)：对知识的深度理解和创造性应用，涉及判断和洞察

## [PARA](PARA.md)：工作流

![CleanShot 2025-03-13 at 00.36.49@2x.png|800](https://imagehosting4picgo.oss-cn-beijing.aliyuncs.com/imagehosting/fix-dir%2Fmedia%2Fmedia_TpNao9Rtdm%2F2025%2F03%2F13%2F00-37-10-e304f777b4d03a9b93cd68a4aebf2686-CleanShot%202025-03-13%20at%2000.36.49-2x-355a9a.png)

1. 捕捉(Capture)
	1. 通过[滴答清单](滴答清单.md)做 inbox 收集闪念笔记，随时记录灵感或者看到好信息的链接，不错过任何灵感火花。
	2. 通过[一切皆项目$](一切皆项目$.md)的项目驱动, 制定方案，吸收信息，项目结束后，信息部分归入 2.1，项目部分归入 2.3
2. 组织(Organize)：将 1 中的信息总结沉淀到[第二大脑](第二大脑.md)的[知识](2%20第二大脑/1%20宇宙概念树/知识.md)中，将信息按主题、项目或领域进行分类整理，建立清晰的知识结构。
3. 提炼(Distill)：将[[2 第二大脑/1 宇宙概念树/知识]]中的内容进一步沉淀到[[飞轮$]]
	1. **[[信息系统]]** - 收集、整理和存储各类数据和信息的基础
	2. **[[认知系统]]** - 处理信息并形成知识和理解的框架
	3. **[[项目系统]]** - 将知识转化为具体行动和项目的执行层
	4. **[[能力系统]]** - 通过项目实践培养和提升的各项能力
	5. **[[影响力系统]]** - 利用能力创造价值并扩大影响范围
	6. **[[投资系统]]** 
4. 表达(Express)：通过[博客](博客.md)的部分，将知识转化为文章、报告或演示，便于分享成型的观点，实现知识的社会化和价值化。
5. [[复盘]]、整个文档做版本控制，基于 commit 的 diff 定期做 AI 复盘

**倒金字塔中部：广泛流行但有一定参与门槛（受众大，有活跃群体，门槛中等）**

- **5\. 深度游戏玩家（米哈游游戏、主机游戏、PC大型网游）**

- **原因**：相比全民手游，这些游戏对设备（高性能手机、主机、PC）和时间投入有更高要求，部分有抽卡等付费机制。如《原神》、《崩坏：星穹铁道》虽流行，但深度参与需要一定的投入。
- （注：部分手游归入此类应该是特指社区里的深渊0t/虚构8W/末日8000强度党或投入较多金钱时间的氪佬，一般人群应该归为第一类的普通手游玩家。）

- **6.音乐技能类（乐器演奏、唱歌训练、作曲基础）**

- **原因**：钢琴、吉他（民谣/电吉他）、架子鼓、小提琴、管乐器、声乐训练、流行歌曲翻唱、简单的编曲/作曲尝试等均属于此类。由于乐器学习普遍，学校会有兴趣班，培训机构也多，虽然精通不易，但入门和兴趣培养的门槛不高。唱歌训练和翻唱更是青少年在KTV、短视频平台常见的娱乐方式。作曲则相对小众但也在兴趣范围内。
- （注：如果你是羽丘/月之森/花咲川的学生，此类爱好（包括组一辈子的乐队）属于第一类极为常见范围。）

- **7.摄影与摄像（手机摄影、Vlog拍摄剪辑）**

- **原因**：随着智能手机大范围普及，使用手机进行日常拍照、学习构图技巧、Vlog（视频日志）的拍摄与剪辑，简单的短片创作等成为常见的宣传自我的方式。短视频平台（抖音、快手、B站直播等）提供了展示和学习的土壤。是记录生活、表达自我和获得关注的流行方式。
- （注：应该指的是经过认真剪辑的Vlog类视频，不包括抖音快手上常见的门槛很低的直播。）

- **8\. 深入二次元（吃谷子、参加漫展、Cosplay、同人创作）**

- **原因**：在观看基础上，更进一步的投入。购买周边（“吃谷子”）需要金钱，参加漫展有时间精力成本，Cosplay和同人创作则需要审美、动手能力、时间和一定财力（服装、道具、绘画工具）。
- （注：我觉得去漫展应该不算要求比较高的，别的如coser、写同人等爱好还是有一定门槛的。）

- **9\. 线上社交与社区（QQ群、Lofter等兴趣社区）**

- **原因**：是上述兴趣的延伸，青少年通过这些平台与同好交流、分享。门槛主要在于活跃度和融入社群。
- （注：指的是在相关社区里高强度分享/写文章等，不是日常聊天。）

- **10\. 普通阅读（名著散文、科幻小说）**

- **原因**：相对网文而言，这类阅读需要更高的阅读门槛和文化积累，受众相对较小，但仍有一定群体。
- （注：泛指阅读除了网络小说以外的传统书籍。）

**倒金字塔下部：小众或专业化（受众极少，参与门槛高，可能需要经济、智力、时间多重投入）**

- **11\. 复杂体育/户外运动（滑旱冰、山地车、攀岩、攀冰、羽毛球进阶、甚至 F1 鉴赏）**

- **原因**：需要专门设备、场地、训练或较高消费。如山地车、攀岩等，门槛较高，F1鉴赏虽然可以看转播，但深入了解和体验的成本不菲。
- （注：F1等专业赛事鉴赏指的是对于赛事有具体了解，只会看到法拉利就来一句“窝法乙烷”而别的一问三不知的梗小鬼不算。）

- **12.交通工具/军事模型爱好者（轨道交通、飞行器、特定军事装备等）**

- **原因**：这类爱好者往往对特定交通工具或军事装备的型号、技术参数、历史沿革、运作原理等有深入的了解和研究，具有一定的“硬核”属性，制作和收藏模型是常见的表现形式；此外，观看并拍摄真实载具是其兴趣的重要组成部分。
- （注：某些火车迷会做出擅闯轨道、攀爬废弃列车场站等高风险行为，应当抵制。）

- **13.进阶艺术与设计（绘画、雕塑、平面设计、数字绘画）**

- **原因**：系统的素描、色彩、油画学习，泥塑、木刻等雕塑，使用PS/AI进行平面设计，或进行数字绘画创作等爱好，都需要专业指导、投入大量时间和金钱（画材、软件、设备）才能达到一定水平，属于更专业化的艺术追求。
- （注：应该指的是在各种平台上发布作品的专业群体。）

- **14.桌面游戏与棋牌进阶（各类桌游、策略棋牌如国际象棋/象棋/围棋高段）**

- **原因**：除了常见的三国杀、狼人杀外，更复杂的策略桌游（如《卡坦岛》、《瘟疫危机》等），以及在国际象棋、象棋、围棋等传统棋类中达到专业或高段水平需要较强的逻辑思维、策略规划能力，以及固定的玩伴或训练对手。专业棋类更是天赋与勤奋的结合。
- （注：类似于围棋高段位等研究深入的选手。）

- **15\. 深度科技与编程（组装电脑、无线电、嵌入式开发、逆向工程、AI训练、数字IC设计）**

- **原因**：对数理基础、逻辑思维、编程能力有极高要求，且往往需要购买专业设备或学习软件，是典型的“硬核”学霸或技术宅的爱好。
- （注：装机我觉得应该不算，这个还是挺常见的；无线电、芯片设计、LLM训练等既需要时间也需要金钱的爱好的确很少见。）

- **16\. 高阶学术竞赛（高联、算法题、数学建模）**

- **原因**：属于学术精英的领域，旨在冲击顶尖大学，门槛在于智力天赋和大量训练时间。
- （注：类似于省一/奥赛金牌的保送大神。）

- **17.进阶语言学习（其他小语种学习）**

- **原因**：除英语、高考日语以外的其他小语种（如德语、法语、韩语、俄语、西班牙语等）的系统学习需要大量时间和毅力，且往往与特定的文化兴趣（如追剧、游戏）或留学规划相关。
- （注：包括为了看番而自学成才到N1的大佬，高考日语必考的除外。）

- **18\. 特定小众文化（听传统戏曲、养鸟、鉴证、深度哲学/历史研究）**

- **原因**：这些爱好可能需要长期的文化熏陶，或是对某些特定领域有超出同龄人的兴趣和钻研。部分“鉴证”可能源于猎奇心理或对复杂政治历史缺乏全面认知，具有一定风险性。
- （注：养鸟戏曲之类的可能是由于参与人数过少而被归入此类；鉴证这里我觉得应该归为中间参与度的部分，毕竟网佐的平均年龄大概率是没有成年的；不过也可能是因为互联网放大了声音，我线下遇到的中学生大部分都是比较正常的。）

- **19\. 投资/金融/高消费嗜好（炒股、HIFI、高端摄影、收藏昂贵手办、奢侈品）**

- **原因**：这些爱好往往与直接的经济投入挂钩，甚至需要专业的金融知识，通常只有在经济条件非常优越的家庭中才能看到。
- （注：不包括股票哥、投资哥等中学非常常见的群体，尤其是下课后喜欢在黑板上用投影装模作样看k线的；除非家里/自己赚的真的有很多。）

- **20\. 极度边缘/争议性爱好（Tulpa、某些极端或负面价值观的“鉴证”行为）**

- **原因**：受众极少，可能涉及心理、伦理、价值观等深层问题，甚至带有争议性。
- （注：大家觉得过于逆天的某些魔怔人属于此类。）

总体看来，大部分高赞下的数学哥、编程哥、文艺哥等等等等，固然不能排除真的出了既有成绩又热爱这门专业的天才，但大部分还是为了装B而装B。
