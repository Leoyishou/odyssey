## 公司内共享的私有的仓库

Nexus 是一个软件仓库管理器,主要用于 Java 开发中。

1. 作为一个私有的 Maven 仓库,可以存放公司内部开Jar 包
2. 可以作为公共仓库的代理,加快依赖下载速度。
3. 帮助管理不同版本的组件。
