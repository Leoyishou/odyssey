## 坦率的建议

- 为探索世界改变世界写代码
- 黑客（hacker）道德（ethic）
- 谦和
- 保持好奇心和进取心
- 喜欢解决问题
- 坚持科学的思维
- 坚持科学的方法
- 积累的时候多注意细节
    - 设身处地为人着想
    - 把自己当用户
- 有些成就之后注意基础
    - 物理是规律
    - 数学真美好
- 做真正的懒人
    - 不喜欢重复
    - 但又要解决问题
    - 积累文档甚至程序自解释
    - 做自己的掘墓人
- 解决问题、不断超越
- 追求效率

自己的风格习惯有意义，但回头看看都是和大原则很相称的，比如代码洁癖，最后会反馈到追求效率上头。
