- 网络（Network）：
    - 监控网络请求和响应
    - 分析页面加载性能
- 元素检查（Elements）：
    - 查看和编辑 HTML 和 CSS
    - 实时修改页面结构和样式
- 控制台（Console）：
    - 执行 JavaScript 代码
    - 查看错误和警告信息
    - 输出调试信息
- 应用（Application）：
    - 管理本地存储、Session 存储、Cookie 等
    - 查看和管理 Service Workers
- 源代码（Sources）：
    - 查看和调试 JavaScript 源码
    - 设置断点进行调试
- 性能（Performance）：
    - 记录和分析网站性能
    - 识别性能瓶颈
- 内存（Memory）：
    - 分析内存使用情况
    - 检测内存泄漏
- 安全（Security）：
    - 查看网站的安全信息
    - 检查证书详情
